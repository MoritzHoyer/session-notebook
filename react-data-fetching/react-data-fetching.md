## Abgerufene Daten mit lokalem Zustand kombinieren

Mit SWR kontrollierst du den Zustand, der die abgerufenen Daten enth√§lt, nicht selbst. Daher kannst du den Zustand nicht direkt √§ndern.       // finde den Film im Zustand
      const info = moviesInfo.find((info) => info.id === id);
